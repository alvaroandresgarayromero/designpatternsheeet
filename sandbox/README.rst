Introduction
--------------

This a sandbox directory that focuses on experimenting
with C++ applications through AWS services. This is not related to design patterns.

- Tools used
    - AWS C++ SDK
    - Terraform
    - Docker

- AWS services used
    - AWS Lambdas (serverless)
    - S3 Bucket
    - IAM roles
